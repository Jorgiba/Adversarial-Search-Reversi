# Adversarial Search AI: Reversi Agent

## Overview
An intelligent agent developed to play **Reversi (Othello)** using adversarial search techniques. The project focuses on decision-making in zero-sum games and heuristic optimization.

## Project Acrhitecture & Credits
This project was developed within a custom Reversi engine provided by the University.
* **My Contribution (strategy.py / reversi_heuristics.py):** I completely designed and implemented the AI logic with my team, including the Minimax algorithm, Alpha-Beta pruning, and the custom heuristic evaluation functions.
* **Provided Framework (board.py, game.py, reversi.py, etc.):** The game loop, board state management, and graphical/console interface were provided as the base engine for the project.

## Implementation Details
* **Algorithm:** Adversarial search strategy (Minimax/Alpha-Beta Pruning).
* **Heuristics:** Custom board evaluation functions based on piece count, corner control, and mobility.
* **Tournament Ready:** The agent was designed to compete in a multi-student tournament, prioritizing efficiency and win rate.
